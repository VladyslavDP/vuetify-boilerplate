# Vuetify Boilerplate

Boilerplate to start with Vuetify

## Steps

1. Fork and clone https://github.com/nimitbhargava/vuetify-boilerplate
2. Add vue router (preferable, not required)
3. `yarn install`
4. `yarn serve`
